下面开始
 当我们需要打造一个校内博客或者公司内部博客再或者是电子公告，而又不让外部看到博客内容的时候，就可以使用 wordpress 来打造一款私密的内部博客了，内部博客就是那种封装性较强的内部网络博客，有点局域网的感觉，不让搜索引擎收录，查看文章要输入密码这个还要把密码告诉别人感觉好麻烦的感觉还是干脆注册才能够查看更叼一些，这个够隐蔽吧，呵呵，看到这里有点网上一些论坛的感觉了吧，好熟悉，哪么我们来看一下利用 wordpress 如何来打造一个私密的博客。

 小小插播一下，上面提到的查看文章输入密码这个在编写文章的时候，在发布中草稿下面有个公开度，可以点击一下设置密码保护或者私密。

 这里主要介绍两种方法，一个是手动改代码，另一个就是插件了，安装插件的方法较为简单一点，打造私密博客的插件是 Private WordPress 了，看这个插件的名称就知道是什么意思了，这里不再多介绍了，安装成功后就必须要注册用户才能够查看文章。

 另一种方法就是不使用插件，我们来手动解决，打造一个内部博客，首先解决的问题就是禁止搜索引擎的抓取，这个很简单，直接一个 roots 文件就可以了，或者在后台，设置-阅读的地方，把建议搜索引擎不索引本站点的勾勾打上即可。

 第二步就是修改模板了，设置网站的内容只允许注册用户才能够查看，未注册的则提示不能查看，修改一切用户能够访问的主题模板，主要是 index.php、category.php、single.php、page.php、archive.php、tag.php、search.php 这几个模板了，在这几个文件的顶部添加如下的代码:


<?php if ( current_user_can("level_0") ) : ?>
